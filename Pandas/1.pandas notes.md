## ðŸ”¹ **4. Data Analysis & Insights**

### **âœ… 4.1 - Total Sales Revenue**
```python
df["Total_Sales"] = df["Price"] * df["Quantity"]
print(df.head())
```

### **âœ… 4.2 - Total Sales per Product**
```python
sales_per_product = df.groupby("Product")["Total_Sales"].sum()
print(sales_per_product)
```

### **âœ… 4.3 - Top 5 Best-Selling Products**
```python
top_products = df.groupby("Product")["Quantity"].sum().sort_values(ascending=False).head(5)
print(top_products)
```

---

## ðŸ”¹ **5. Date-Based Analysis**
If you want to analyze sales **over time**, use the `datetime` format.

### **âœ… Convert Date Column to DateTime**
```python
df["Date"] = pd.to_datetime(df["Date"])
```

### **âœ… Sales Trend Over Time**
```python
sales_trend = df.groupby(df["Date"].dt.month)["Total_Sales"].sum()
print(sales_trend)
```
